PROJETO ESTÁGIO

Instruções

Coloque os lixos caindo do céu no lugar certo! Ao colocar os 5 itens no seu devido lugar, você ganhará! Utilize as setas de seu teclado para guiar o lixo para o caminho correto! Caso você erre, clique na bandeira para recomeçar.

Cŕeditos

Projeto criado pelo professor Guilherme Coelho. Acesse a playlist de vídeo-aulas para criá-lo do zero!

PROPÓSITO

O propósito deste pequeno jogo é, através de lógica de programação, enraizar a reciclagem, apresentando diversos tipos de lixos e seus respectivos recipientes.
